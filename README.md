Judul : Inventory App

Database : Firebase

Fungsi yang ada : 
- CRUD stok barang
- Stok barang berisi : nama barang, foto barang, jenis barang, detail barang, jumlah barang, supplier
- Push notification

Widget yang akan digunakan : list view builder, snackbar, expanded, dismissible, autocomplete, dropdown, slideable
